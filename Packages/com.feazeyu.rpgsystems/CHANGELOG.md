# Changelog

All notable changes to this package are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres
to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-28

### Added

- Initial release.
- Unified graph framework: graph assets, runner with parallel flow tokens, blackboard, and a reusable graph editor window.
- Dialogue system: branching lines, conditional choices, requirement gating, and text interpolation.
- Quest system: single quests (objectives, rewards, terminal outcomes) and quest chains with gates and per-quest variables; composable objectives with gate, timer, and reset modifiers.
- Inventory system: shape-aware grid inventory, linear list inventory, and a shop layer.
- Character/combat layer: entity resources, stat/effect scaling, weapons, and an interaction system.
- Dialogue, Quest, and Inventory samples.
