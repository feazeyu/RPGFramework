\doxysubsection{Detailed Description}
Handler for the Collect Item objective node (type\+Id = "{}obj\+\_\+collect"{}). 

{\bfseries{Level-\/based}}:\+ completes once the linked \doxylink{namespace_feazeyu_1_1_r_p_g_systems_1_1_inventory}{Inventory} container holds at least Count of Item\+Id — it checks the absolute amount, so whatever the player already carried counts. For "{}acquire N from now on, ignoring the starting amount"{} (and gate/\+timer support) use {\bfseries{Accumulate Item}}.

Waits until the container has Count of Item\+Id, then follows "{}\+Completed"{}. For a "{}while carrying X"{} constraint, attach an Item Gate to the guarded objective\textquotesingle{}s In port instead.
