\doxysubsection{Detailed Description}
\hypertarget{dir_8ef45574f42f7d0335f5ddb2d0096129_autotoc_md3}{}\doxysubsection{\texorpdfstring{RPG Systems}{RPG Systems}}\label{dir_8ef45574f42f7d0335f5ddb2d0096129_autotoc_md3}
Graph-\/based {\bfseries{dialogue}} and {\bfseries{quest}} systems, a flexible {\bfseries{inventory}} (grid, list, and shop), and a small {\bfseries{character/\+combat}} layer for 2D RPGs in Unity. The dialogue and quest systems share a single graph execution framework, so custom nodes and blackboard variables work the same across both.\hypertarget{dir_8ef45574f42f7d0335f5ddb2d0096129_autotoc_md4}{}\doxysubsection{\texorpdfstring{Features}{Features}}\label{dir_8ef45574f42f7d0335f5ddb2d0096129_autotoc_md4}

\begin{DoxyItemize}
\item {\bfseries{Unified graph framework}} — node/\+edge/\+blackboard model with a coroutine-\/driven runner, parallel flow tokens, subgraphs, and a full graph editor window (canvas, blackboard, inspector) you can reuse for your own graph-\/based systems.
\item {\bfseries{Dialogue}} — branching lines, conditional choices, requirement gating, text interpolation from blackboard variables, and inline event/\+prefab actions.
\item {\bfseries{Quests}} — single quests (objectives, rewards, terminal outcomes) and quest chains (dependency graphs with gates and per-\/quest variables). Composable objectives (kill count, collect/\+accumulate/\+deliver items, reach location) with stackable gate, timer, and reset modifiers.
\item {\bfseries{Inventory}} — shape-\/aware grid inventory with drag-\/and-\/drop and stacking, a linear list inventory, and a shop layer (buy/\+sell against a runtime copy of the stock).
\item {\bfseries{Character \& combat}} — entity resources (health/\+mana/\+stamina), a stat/\+effect scaling system, weapons (melee, projectile, combos), and an interaction system.
\end{DoxyItemize}\hypertarget{dir_8ef45574f42f7d0335f5ddb2d0096129_autotoc_md5}{}\doxysubsection{\texorpdfstring{Requirements}{Requirements}}\label{dir_8ef45574f42f7d0335f5ddb2d0096129_autotoc_md5}
