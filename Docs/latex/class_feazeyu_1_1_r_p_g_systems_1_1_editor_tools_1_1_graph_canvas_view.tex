\doxysubsection{Detailed Description}
The shared graph canvas. Extends Unity\textquotesingle{}s Graph\+View so we get node drag, rubber-\/band selection, edge drawing, pan/\+zoom, and minimap for free. 

System-\/agnostic — used by every graph-\/based editor (dialogue, quest, ...). What makes it system-\/specific is the three constructor parameters:\+ 
\begin{DoxyItemize}
\item {\itshape node\+Registry}  — populates the "{}\+Add Node"{} context menu 
\item {\itshape theme\+Sheet}  — system-\/specific colour USS layered over Graph\+Editor.\+uss 
\item {\itshape root\+Css\+Class}  — CSS class added to {\ttfamily this} so the theme sheet\textquotesingle{}s {\ttfamily .xxx-\/graph-\/view} rules apply 
\end{DoxyItemize}
