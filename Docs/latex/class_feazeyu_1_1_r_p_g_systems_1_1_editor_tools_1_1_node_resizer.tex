\doxysubsection{Detailed Description}
Adds resize handles to a Visual\+Element (the node card). 

Seven hit zones are overlaid on the node edges and corners:\+

Only right, bottom and the SE corner are exposed — top/\+left would fight with the Graph\+View\textquotesingle{}s own drag-\/to-\/move gesture.

Usage:\+ target.\+Add\+Manipulator(new Node\+Resizer(on\+Resized));
