\doxysubsection{Detailed Description}
A single quest reference inside a chain. 

Source indicates which of Graph\+Quest /\+ Simple\+Quest is populated — exactly one is non-\/null for a well-\/formed entry. None means the link was missing or pointed at an unsupported type.
