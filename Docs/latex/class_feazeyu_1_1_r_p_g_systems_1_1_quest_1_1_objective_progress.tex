\doxysubsection{Detailed Description}
Resettable progress counter shared between an objective handler and any modifiers (gates, timers, reset nodes) attached to that objective node. 

The objective handler raises raw events via Try\+Add; the composed Gate predicate filters them, so an event only counts when every attached gate is satisfied. A Timer/\+\+Reset node can wipe accumulated progress at any time via Reset.
