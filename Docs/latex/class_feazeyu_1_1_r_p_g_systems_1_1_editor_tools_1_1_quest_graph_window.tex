\doxysubsection{Detailed Description}
Quest-\/system graph editor window. Sibling of Dialogue\+Graph\+Window; inherits from Graph\+Editor\+Window and differs in:\+ • the palette source (Quest\+Node\+Registry) • the theme sheet ({\ttfamily Quest\+Graph.\+uss}) • the CSS class ({\ttfamily quest-\/graph-\/view}) • the asset factory (Quest\+Graph\+Asset). 

Unique behaviour:\+ the palette filters by Quest\+Graph\+Asset.\+Kind. A Quest\+Kind.\+Single asset sees the Objective/\+\+Reward/\+\+Complete/\+\+Fail palette; a Quest\+Kind.\+Chain asset sees Run\+Subgraph + flow nodes only. The filter runs on every Graph\+Editor\+Window.\+Load\+Asset via the Graph\+Editor\+Window.\+Get\+Node\+Registry\+For\+Asset hook.
