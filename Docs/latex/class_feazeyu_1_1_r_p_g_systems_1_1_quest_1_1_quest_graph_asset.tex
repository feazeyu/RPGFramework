\doxysubsection{Detailed Description}
The quest-\/system Scriptable\+Object asset. Sibling of Dialogue\+Graph\+Asset; both inherit from Graph\+Asset to share data and CRUD. 

Two entries appear under {\bfseries{Assets → Create → \doxylink{namespace_feazeyu_1_1_r_p_g_systems_1_1_quest}{Quest}}}:\+ 
\begin{DoxyItemize}
\item {\bfseries{\doxylink{namespace_feazeyu_1_1_r_p_g_systems_1_1_quest}{Quest} Graph}} — creates a Single quest 
\item {\bfseries{\doxylink{namespace_feazeyu_1_1_r_p_g_systems_1_1_quest}{Quest} Chain}} — creates a Chain graph (still a \doxylink{class_feazeyu_1_1_r_p_g_systems_1_1_quest_1_1_quest_graph_asset}{Quest\+Graph\+Asset}, with Kind pre-\/set to Quest\+Kind.\+Chain) 
\end{DoxyItemize}The kind is serialised on the asset; the editor\textquotesingle{}s node palette and the runtime behaviour both switch on it.
