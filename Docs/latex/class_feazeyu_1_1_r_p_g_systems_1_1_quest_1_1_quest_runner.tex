\doxysubsection{Detailed Description}
Executes a Single quest graph. Supports any number of simultaneously active objectives (needed for continuous-\/mode parallel objectives). 

Objective node handlers call Register\+Objective /\+ Unregister\+Objective directly; external code still calls Complete\+Objective /\+ Fail\+Objective by GUID as before.
