\doxysubsection{Detailed Description}
Subclass of \doxylink{class_feazeyu_1_1_r_p_g_systems_1_1_dialogue_1_1_graph_runner}{Graph\+Runner} that handles dialogue-\/specific node types:\+ • Dialogue\+Line — fires On\+Dialogue\+Line, waits for \doxylink{class_feazeyu_1_1_r_p_g_systems_1_1_dialogue_1_1_dialogue_runner_a11fc763156c9d43ae4602100700e9226}{Advance()} • Choice\+Branch — fires On\+Choices\+Presented, waits for \doxylink{class_feazeyu_1_1_r_p_g_systems_1_1_dialogue_1_1_dialogue_runner_ad27cc7617d45accaf4a8d8cb99a00713}{Select\+Choice()} • Trigger\+Event — fires On\+Event\+Triggered • Wait\+For\+Event — stubbed (logs a warning). 

All structural nodes (Start, End, Condition, Set\+Variable, Run\+Subgraph) are handled by the base \doxylink{class_feazeyu_1_1_r_p_g_systems_1_1_dialogue_1_1_graph_runner}{Graph\+Runner}.
