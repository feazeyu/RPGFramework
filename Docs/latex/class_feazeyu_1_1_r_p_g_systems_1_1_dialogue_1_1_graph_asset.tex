\doxysubsection{Detailed Description}
Common serialised data and CRUD for every graph-\/based system (\doxylink{namespace_feazeyu_1_1_r_p_g_systems_1_1_dialogue}{Dialogue}, \doxylink{namespace_feazeyu_1_1_r_p_g_systems_1_1_quest}{Quest}, and any future graph editor built on top of Feazeyu.\+RPGSystems.\+Editor\textquotesingle{}s window pipeline). 

Concrete subclasses (Dialogue\+Graph\+Asset, Quest\+Graph\+Asset) exist only to carry a {\ttfamily \+[Create\+Asset\+Menu]\+} attribute so each system gets its own entry under {\bfseries{Assets → Create}}.

Serialisation layout — {\bfseries{do not change}} without a migration step. Feazeyu.\+RPGSystems.\+Editor\+Tools.\+Blackboard\+Property\+Bridge walks {\ttfamily m\+\_\+\+Blackboard.\+m\+\_\+\+Variables} via absolute Serialized\+Property paths; any rename here silently breaks the inspector\textquotesingle{}s bound fields.
