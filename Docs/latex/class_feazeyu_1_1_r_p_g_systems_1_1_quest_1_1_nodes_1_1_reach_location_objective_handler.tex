\doxysubsection{Detailed Description}
Handler for the Reach Location objective node (type\+Id = "{}obj\+\_\+location"{}). 

Waits until the player is within Radius of Target, then follows "{}\+Completed"{}. Use this to gate the next objective behind arriving somewhere — connect Completed → next objective\textquotesingle{}s In. For a "{}do X while staying in zone Y"{} constraint, attach a Location Gate to the guarded objective\textquotesingle{}s In port instead.

{\bfseries{Target resolution}} (first match wins):\+
\begin{DoxyEnumerate}
\item \doxylink{class_feazeyu_1_1_r_p_g_systems_1_1_dialogue_1_1_blackboard}{Blackboard} variable of type Transform linked to the Target field.
\item Inline value treated as a scene object name (Game\+Object.\+Find). 
\end{DoxyEnumerate}
