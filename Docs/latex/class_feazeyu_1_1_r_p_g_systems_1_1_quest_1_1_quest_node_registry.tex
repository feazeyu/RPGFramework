\doxysubsection{Detailed Description}
Node registry for the \doxylink{namespace_feazeyu_1_1_r_p_g_systems_1_1_quest}{Quest} graph system. Derives from Node\+Registry, so the shared flow/\+logic nodes come from Node\+Registry.\+Register\+Common\+Nodes; this class adds the quest-\/specific palette (Objective, Reward, terminals, objectives) and the quest flavour of Condition and \doxylink{namespace_feazeyu_1_1_r_p_g_systems_1_1_quest}{Quest} Reference (Run Subgraph). Contains every node type either flavour of quest graph can use; the window filters by Quest\+Graph\+Asset.\+Kind via For\+Kind. 

Type-\/id constants for shared flow/\+logic nodes mirror Node\+Registry so a node\textquotesingle{}s stored {\ttfamily Node\+Type} string is universal — a "{}\+Start"{} node means the same thing regardless of which graph system it lives in.
