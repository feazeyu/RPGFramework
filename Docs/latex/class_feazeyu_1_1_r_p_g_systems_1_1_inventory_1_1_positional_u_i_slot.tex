\doxysubsection{Detailed Description}
A reusable UI cell that bridges a single drawn slot to an IUIPositional\+Item\+Container. It is both the drag source (ISingle\+Item\+Container) and the drop target (IDrop\+Handler) for the item rendered in that cell, so any inventory — grid, list, or a custom layout — gets drag-\/and-\/drop by attaching one of these per cell. 

Wire it in code (the target field is an interface and so is not Inspector-\/serializable):\+ add the component, then set target to your container and position to the cell\textquotesingle{}s address. Build the item visual with Inventory\+Helper.\+Create\+UIDrag\+Handler and ensure a Drag\+Layer exists via Inventory\+Helper.\+Generate\+Drag\+Layer.
