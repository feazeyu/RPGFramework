\doxysubsection{Detailed Description}
Single-\/\+Mono\+Behaviour quest HUD for the chain demo. 

Binds to the Quest\+Chain\+Runner rather than a fixed Quest\+Runner, because chain quests spawn a fresh runner per quest at runtime. As each quest starts it latches onto its runner via Quest\+Chain\+Runner.\+Get\+Active\+Runner and mirrors that runner\textquotesingle{}s objective events into a checklist, appending the live (current/\+required) counter for accumulation objectives (Kill Count, Accumulate \doxylink{class_feazeyu_1_1_r_p_g_systems_1_1_items_1_1_item}{Item}, …) straight from Quest\+Runner.\+Get\+Progress.
