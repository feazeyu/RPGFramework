\doxysubsection{Detailed Description}
Handler for the Deliver Item objective node (type\+Id = "{}obj\+\_\+deliver"{}). 

Waits for the player to interact with the specified NPC while the linked \doxylink{namespace_feazeyu_1_1_r_p_g_systems_1_1_inventory}{Inventory} contains at least Count of Item\+Id. On success the items are removed and the node follows "{}\+Completed"{}.

{\bfseries{Field resolution}} (first match wins):\+ NPC — blackboard Game\+Object variable, or inline scene object name (Game\+Object.\+Find). \doxylink{namespace_feazeyu_1_1_r_p_g_systems_1_1_inventory}{Inventory} — blackboard Game\+Object variable with an \doxylink{interface_feazeyu_1_1_r_p_g_systems_1_1_inventory_1_1_i_item_container}{IItem\+Container} component.

The NPC\textquotesingle{}s \doxylink{class_feazeyu_1_1_r_p_g_systems_1_1_character_1_1_interactable_ab236bf4b0418eff83211a852c759ad07}{Interactable.\+On\+Interact} event is used for delivery — make sure the NPC has an \doxylink{class_feazeyu_1_1_r_p_g_systems_1_1_character_1_1_interactable}{Interactable} component.
