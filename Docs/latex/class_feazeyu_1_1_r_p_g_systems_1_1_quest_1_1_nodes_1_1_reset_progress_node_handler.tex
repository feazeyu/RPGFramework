\doxysubsection{Detailed Description}
When a token reaches this node, it wipes the accumulated progress of every objective it targets, then continues out {\ttfamily Out}. 

Targets are chosen by edge:\+ wire this node\textquotesingle{}s {\ttfamily Target} output into the {\ttfamily In} port of each objective whose progress should reset. Target edges are reference-\/only — they never re-\/enter or re-\/run the objective.

Compose with a Timer (Timeout → Begin … Timeout → Reset Progress) for "{}reset on expiry"{}, or trigger from any flow path for manual resets.
