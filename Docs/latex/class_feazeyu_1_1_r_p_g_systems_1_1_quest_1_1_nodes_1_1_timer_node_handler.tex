\doxysubsection{Detailed Description}
A standalone timer. A token entering the {\ttfamily Begin} input starts the clock; after {\ttfamily Seconds} it fires Quest\+Runner.\+On\+Timer\+Timeout and forks a token down {\ttfamily Timeout}. With {\ttfamily Auto\+Restart} on it re-\/arms and repeats as a perpetual watchdog (forking a fresh token each cycle while keeping its own token alive); off, it times out once. 

The timer has no built-\/in side effect â€” wire {\ttfamily Timeout} to whatever should happen (e.\+g. a Reset Progress node) to compose behaviour.
