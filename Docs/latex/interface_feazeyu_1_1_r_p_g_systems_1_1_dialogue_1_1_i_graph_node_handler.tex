\doxysubsection{Detailed Description}
Implement this interface to define what happens when the graph runner reaches a node of a specific type. 

Register implementations with \doxylink{class_feazeyu_1_1_r_p_g_systems_1_1_dialogue_1_1_graph_runner_a7996fdb2293cb06f7df139785acdb4a0}{Graph\+Runner.\+Register\+Handler()}, or place a \doxylink{class_feazeyu_1_1_r_p_g_systems_1_1_dialogue_1_1_graph_node_behaviour}{Graph\+Node\+Behaviour} component on the runner\textquotesingle{}s Game\+Object so it is discovered automatically.

When execution is complete the handler must call either:\+ context.\+Follow("{}\+Port\+Name"{}) — advance along an output port context.\+End() — terminate the graph Failing to call either will stall the graph indefinitely.
