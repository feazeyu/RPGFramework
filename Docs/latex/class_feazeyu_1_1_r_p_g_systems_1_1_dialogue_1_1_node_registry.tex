Abstract base for every graph system\textquotesingle{}s node registry. Provides the shared infrastructure all registries need:\+ 
\begin{DoxyItemize}
\item The universal flow/\+logic node type-\/id constants and accent colours, so a stored {\ttfamily Node\+Type} string means the same thing in any graph system (a "{}\+Start"{} node is a "{}\+Start"{} node everywhere). 
\item The lazily-\/built type lookup table plus the Register /\+ All\+Nodes /\+ Get\+Node plumbing. 
\item Register\+Common\+Nodes — the flow/\+logic/\+scene nodes shared by every system — and Register\+Attribute\+Nodes$<$\+TAttr$>$ for reflection-\/based discovery. 
\end{DoxyItemize}Concrete registries ({\ttfamily \doxylink{class_feazeyu_1_1_r_p_g_systems_1_1_dialogue_1_1_dialogue_node_registry}{Dialogue\+Node\+Registry}}, {\ttfamily Quest\+Node\+Registry}) derive from this, override Build to add their own node set, and expose a static façade over a private singleton instance.
