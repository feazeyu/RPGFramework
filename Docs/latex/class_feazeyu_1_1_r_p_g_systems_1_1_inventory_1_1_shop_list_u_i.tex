\doxysubsection{Detailed Description}
A shop rendered as a scrollable list. It is an Inventory\+List pre-\/stocked from a Shop\+Inventory, so it reuses the standard list rendering and drag-\/and-\/drop (Inventory\+List\+Generator /\+ Inventory\+List\+UI) — mirroring how Shop\+Grid\+UI builds on Inventory\+Grid. 

A listing\textquotesingle{}s stock is its slot\textquotesingle{}s stack depth:\+ dragging an item out buys one (charging the player), dragging an item in sells it back (paying the player). Infinite-\/stock listings never deplete and render as "{}∞"{}. Each instance runs against a Shop\+Inventory.\+Clone\+For\+Runtime copy, so buying/\+selling never mutates the authored asset.

Requires an Inventory\+List\+Generator on the same Game\+Object (added automatically via the base On\+Validate); set its slot\+Prefab (with a Text\+Count\+Item\+Renderer) and canvas.
