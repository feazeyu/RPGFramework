\doxysubsection{Detailed Description}
Shared base for every graph-\/editor window (dialogue, quest, and anything built on top of the Graph\+Asset pipeline). 

Responsibilities:\+ • Build the toolbar /\+ blackboard /\+ canvas /\+ inspector layout • Load the base {\ttfamily Graph\+Editor.\+uss} stylesheet • Wire node-\/selection events between the canvas and inspector • Handle asset New /\+ Open /\+ Save /\+ Auto-\/\+Save

Subclasses supply what\textquotesingle{}s system-\/specific via the template-\/method properties at the top:\+ 
\begin{DoxyItemize}
\item Window\+Title — window title string 
\item Pref\+Key\+Prefix — Editor\+Prefs namespace for last-\/asset memory 
\item New\+Asset\+Name, Save\+Dialog\+Title, Window\+Icon 
\item Node\+Registry\+Source — the "{}\+Add Node"{} palette 
\item Theme\+Style\+Sheet — system-\/specific colour overrides 
\item Graph\+View\+Css\+Class — root class added to the canvas (drives theme rules) 
\item Create\+Asset\+Instance — factory for the right subclass of Graph\+Asset 
\end{DoxyItemize}
