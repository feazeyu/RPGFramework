Detailed Description

Drag handler for a non-anchor (satellite) cell of a multi-cell item. Resolves the real owning slot through targetPosition so the drag originates from, and drops resolve to, the item's anchor cell. Use this instead of InventoryItemUIHandler on satellite cells of shaped items; InventoryHelper.CreateUIDragHandler attaches it when redirector: true.
