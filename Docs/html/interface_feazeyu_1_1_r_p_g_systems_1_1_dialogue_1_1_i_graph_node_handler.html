Detailed Description

Implement this interface to define what happens when the graph runner reaches a node of a specific type.

Register implementations with GraphRunner.RegisterHandler(), or place a GraphNodeBehaviour component on the runner's GameObject so it is discovered automatically.

When execution is complete the handler must call either: context.Follow("PortName") — advance along an output port context.End() — terminate the graph Failing to call either will stall the graph indefinitely.
