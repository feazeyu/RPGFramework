◆ QuestKind

enum Feazeyu.RPGSystems.Quest.QuestKind

Which flavour of quest graph this asset represents.

Single — the quest's internal logic. Uses Objective / Reward / CompleteQuest / FailQuest nodes. Executed by QuestRunner like any graph.

Chain — a dependency graph of quests. Each RunSubgraph node references a Single quest asset; edges between RunSubgraph nodes mean prerequisite ("B depends on A"). Not walked linearly — QuestChainRunner maintains completion state and exposes the topological frontier as AvailableQuests.
