Detailed Description

Quest-system graph editor window. Sibling of DialogueGraphWindow; inherits from GraphEditorWindow and differs in: • the palette source (QuestNodeRegistry) • the theme sheet (QuestGraph.uss) • the CSS class (quest-graph-view) • the asset factory (QuestGraphAsset).

Unique behaviour: the palette filters by QuestGraphAsset.Kind. A QuestKind.Single asset sees the Objective/Reward/Complete/Fail palette; a QuestKind.Chain asset sees RunSubgraph + flow nodes only. The filter runs on every GraphEditorWindow.LoadAsset via the GraphEditorWindow.GetNodeRegistryForAsset hook.
