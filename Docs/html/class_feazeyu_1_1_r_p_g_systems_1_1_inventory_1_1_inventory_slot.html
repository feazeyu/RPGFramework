virtual bool Feazeyu.RPGSystems.Inventory.InventorySlot.AcceptsItem ( ItemInfo item)

Determines whether the slot can accept the specified item. Override this to filter by item type, category, ItemTarget, etc. The grid placement path and the base PutItem(GameObject) both call this overload, so a filter declared here is enforced at placement time. Defaults to the item-agnostic AcceptsItem() check so existing slots keep their behavior.
