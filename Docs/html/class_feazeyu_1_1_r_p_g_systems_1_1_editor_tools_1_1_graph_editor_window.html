Detailed Description

Shared base for every graph-editor window (dialogue, quest, and anything built on top of the GraphAsset pipeline).

Responsibilities: • Build the toolbar / blackboard / canvas / inspector layout • Load the base GraphEditor.uss stylesheet • Wire node-selection events between the canvas and inspector • Handle asset New / Open / Save / Auto-Save

Subclasses supply what's system-specific via the template-method properties at the top:

  • WindowTitle — window title string
  • PrefKeyPrefix — EditorPrefs namespace for last-asset memory
  • NewAssetName, SaveDialogTitle, WindowIcon
  • NodeRegistrySource — the "Add Node" palette
  • ThemeStyleSheet — system-specific colour overrides
  • GraphViewCssClass — root class added to the canvas (drives theme rules)
  • CreateAssetInstance — factory for the right subclass of GraphAsset
