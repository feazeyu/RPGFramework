Detailed Description

A shop rendered as a scrollable list. It is an InventoryList pre-stocked from a ShopInventory, so it reuses the standard list rendering and drag-and-drop (InventoryListGenerator / InventoryListUI) — mirroring how ShopGridUI builds on InventoryGrid.

A listing's stock is its slot's stack depth: dragging an item out buys one (charging the player), dragging an item in sells it back (paying the player). Infinite-stock listings never deplete and render as "∞". Each instance runs against a ShopInventory.CloneForRuntime copy, so buying/selling never mutates the authored asset.

Requires an InventoryListGenerator on the same GameObject (added automatically via the base OnValidate); set its slotPrefab (with a TextCountItemRenderer) and canvas.
