Detailed Description

Executes a Single quest graph. Supports any number of simultaneously active objectives (needed for continuous-mode parallel objectives).

Objective node handlers call RegisterObjective / UnregisterObjective directly; external code still calls CompleteObjective / FailObjective by GUID as before.
