 CNodeRegistryAbstract base for every graph system's node registry. Provides the shared infrastructure all registries need:

  • The universal flow/logic node type-id constants and accent colours, so a stored NodeType string means the same thing in any graph system (a "Start" node is a "Start" node everywhere).
  • The lazily-built type lookup table plus the Register / AllNodes / GetNode plumbing.
  • RegisterCommonNodes — the flow/logic/scene nodes shared by every system — and RegisterAttributeNodes<TAttr> for reflection-based discovery.

Concrete registries (DialogueNodeRegistry, QuestNodeRegistry) derive from this, override Build to add their own node set, and expose a static façade over a private singleton instance
