Detailed Description

A standalone timer. A token entering the Begin input starts the clock; after Seconds it fires QuestRunner.OnTimerTimeout and forks a token down Timeout. With AutoRestart on it re-arms and repeats as a perpetual watchdog (forking a fresh token each cycle while keeping its own token alive); off, it times out once.

The timer has no built-in side effect — wire Timeout to whatever should happen (e.g. a Reset Progress node) to compose behaviour.
