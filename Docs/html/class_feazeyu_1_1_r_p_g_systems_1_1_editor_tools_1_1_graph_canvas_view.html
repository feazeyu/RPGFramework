Detailed Description

The shared graph canvas. Extends Unity's GraphView so we get node drag, rubber-band selection, edge drawing, pan/zoom, and minimap for free.

System-agnostic — used by every graph-based editor (dialogue, quest, ...). What makes it system-specific is the three constructor parameters:

  • nodeRegistry — populates the "Add Node" context menu
  • themeSheet — system-specific colour USS layered over GraphEditor.uss
  • rootCssClass — CSS class added to this so the theme sheet's .xxx-graph-view rules apply
