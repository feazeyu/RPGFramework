Detailed Description

A shop that renders exactly like InventoryGrid. Items are pre-populated from a ShopInventory. Dragging an item out charges the player via IShopCurrency; if they cannot afford it, the drag is blocked. Requires an InventoryGridGenerator on the same GameObject (added automatically via OnValidate).
