Detailed Description

Optional capability for an IItemContainer that announces content changes. Lets observers (e.g. an Accumulate Item quest objective) count items gained and lost exactly, instead of polling IItemContainer.CountItem. Tracking both directions makes "net acquired" robust: dropping then re-collecting nets to one count.
