Detailed Description

A reusable UI cell that bridges a single drawn slot to an IUIPositionalItemContainer. It is both the drag source (ISingleItemContainer) and the drop target (IDropHandler) for the item rendered in that cell, so any inventory — grid, list, or a custom layout — gets drag-and-drop by attaching one of these per cell.

Wire it in code (the target field is an interface and so is not Inspector-serializable): add the component, then set target to your container and position to the cell's address. Build the item visual with InventoryHelper.CreateUIDragHandler and ensure a DragLayer exists via InventoryHelper.GenerateDragLayer.
