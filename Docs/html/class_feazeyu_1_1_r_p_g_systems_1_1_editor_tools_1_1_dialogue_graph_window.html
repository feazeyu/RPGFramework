Detailed Description

Dialogue-system graph editor window.

All layout, toolbar, blackboard and inspector UI — including the stylesheet loading pipeline — lives in GraphEditorWindow. This subclass only supplies the dialogue-specific window title, node palette, theme sheet, CSS class, asset factory and menu hooks.
