Detailed Description

Adds resize handles to a VisualElement (the node card).

Seven hit zones are overlaid on the node edges and corners:

Only right, bottom and the SE corner are exposed — top/left would fight with the GraphView's own drag-to-move gesture.

Usage: target.AddManipulator(new NodeResizer(onResized));
